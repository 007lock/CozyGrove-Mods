# Simon's Cozy Grove Mods

Only usable for the Windows version of Cozy Grove.

## Mods

### ToggleInventory

Toggles the player's inventory. The default hotkey is `Tab`.

### PreventHUDFade

Makes it so the UI never fades when running. If, for some reason, it begins to fade again, the default hotkey to toggle on the prevention is `F10`. If you would prefer to turn the UI on and off when desired, then be sure to checkout [Gibbed's mod](https://github.com/gibbed/CozyGrove-Mods).

### AutoHarvest

Automatically harvests collectables and doobers. Default hotkey is `Ctrl + H`.

### AutoNet

Automatically catch critters. Default hotkey is `Ctrl + T`.

### AutoFishing

Automatically catch fish. Default hotkey is `Ctrl + F`.

### SkippingShell

Automatically finding nearest Opened Shell and skip stones to it. Default hotkey is `Ctrl + S`.

## Installation (Windows)

1. Install [MelonLoader](https://github.com/LavaGang/MelonLoader/releases) **v0.7.2+** to Cozy Grove.
2. Download the the mod of your choice from [the latest release](https://github.com/007lock/CozyGrove-Mods/releases/latest).
3. Extract the mod DLL (such as `Ashnel.CozyGarden.ToggleInventory.dll`) to the `Mods` folder in Cozy Grove's installation location.
   - If the `Mods` folder does not exist, create it.
